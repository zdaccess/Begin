1. Команда создает автоматически папку target в корне проекта и подпапки в ней. Компилирует файлы в конечную папку target/edu/school21/printer

javac -d target src/java/edu/school21/printer/*/*.java


2. Команда запускает программу файлы которой находятся по адресу target/edu/school21/printer, в конце необходимо через пробел указать три аргумента:
    a) символ, который будет показывать белый цвет;
    б) символ, который будет показывать черный цвет;
    в) абсолютный путь к файлу bmp на диске.

java -cp target edu.school21.printer.Program . 0 absolute_path.bmp
